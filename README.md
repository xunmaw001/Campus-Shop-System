# Campus-Shop-System
基于微信小程序的校园商铺系统，为校园师生带来便捷购物体验。该系统集合各类校园商铺，商品涵盖零食、文具、生活用品等。师生无需到店，在小程序上即可浏览商品详情、查看价格与库存。下单后，支持配送上门或到店自提。同时，商铺可在后台管理商品、订单和客户信息，开展促销活动。系统还设有评价功能，方便师生反馈购物感受，促进商铺改进服务。有了它，校园购物更高效、轻松，提升师生生活便利性。
